## como clonar um repositório

1- Passo a passo:

```bash
# Navegando até o disco C
cd /c
# criando uma pasta
mkdir minha-pasta
#entrando na pasta 
cd minha-pasta

#Clonando repositório
git clone <link-repositorio>

#acessando o repositório
cd <path-repositório>

```
